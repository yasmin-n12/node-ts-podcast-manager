# 🎙️ Gerenciador de Podcasts API

API REST desenvolvida em Node.js com TypeScript utilizando estritamente o módulo nativo `http` do Node, sem o uso de frameworks externos (como Express). Projeto realizado como desafio prático para a plataforma DIO.

## 🛠️ Tecnologias Utilizadas
- **Node.js** (Ambiente de execução)
- **TypeScript** (Tipagem estática)
- **HTTP Module** (Módulo nativo do Node.js)
- **ts-node-dev** (Ambiente de desenvolvimento com live-reload)
